## Developing

### Prerequisites

- Node 9+
- NPM 5.6+
- Angular CLI
- Typescript
- Covalent 2.0+
- TSLint

### Built With

Recommand yarn instead of npm, specially when you install node_modules

```shell
npm i -g yarn@latest
```

Install Angular CLI

```shell
npm i -g @angular/cli@latest
```

Install Typescript and tslint

```shell
npm i -g typescript
npm i -g tslint
```

Install node_modules

```shell
npm install | yarn install
```
